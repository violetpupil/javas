# [lifecycle](https://maven.apache.org/guides/introduction/introduction-to-the-lifecycle.html)

Maven的生命周期由一系列阶段（phase）构成

指定阶段后，Maven 自动识别对应的生命周期，并运行到对应阶段

阶段最终由插件的 goal 执行

## clean

- `clean` remove all files generated by the previous build

## default

- `compile` 编译类文件
- `test` 运行测试
- `package` 打包
