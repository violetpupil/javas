# [Maven](https://maven.apache.org/)

[Maven Wrapper](https://maven.apache.org/wrapper/)

## 项目结构

- pom.xml 项目描述文件
- src
  - main
    - java 程序代码
  - test
    - java 测试代码
- target 编译打包生成文件

## mvn

`-version` 版本

`-P` Comma-delimited list of profiles to activate

`--quiet` 简化输出

`-X` 输出调试信息
