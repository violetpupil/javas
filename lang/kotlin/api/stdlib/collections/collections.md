# [kotlin.collections](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin.collections/)

## Types

List 只读列表

Map 只读映射

MutableList 读写列表

MutableMap 读写映射

MutableSet 读写无序集合

Set 只读无序集合

## Functions

listOf() 创建只读列表

mapOf() 创建只读映射

mutableListOf() 创建读写列表

mutableMapOf() 创建读写映射

mutableSetOf() 创建读写无序集合

setOf() 创建只读无序集合
